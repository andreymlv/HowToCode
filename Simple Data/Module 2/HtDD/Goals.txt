# Цели

- [] Уметь использовать рецепт How to Design Data Definitions (HtDD) для разработки определений данных для атомарных данных.
- [] Уметь определять информацию предметной области, которая должна быть представлена в виде простых атомарных данных, интервалов, перечислений, элементов и смешанных элементов данных.
- [] Уметь использовать рецепт Data Driven Templates для создания шаблонов для функций, работающих с атомарными данными.
- [] Уметь использовать рецепт How to Design Functions (HtDF) для проектирования функций, работающих с атомарными данными.
